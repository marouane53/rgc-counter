# Legacy Launchers

These wrappers are archived from the earlier Cell Counting Project era and are no longer the recommended entrypoint for `retinal-phenotyper`.

Supported entrypoints:
- CLI runs: `python main.py ...` from a checkout or `retinal-phenotyper ...` from an installed package
- Local UI: `python scripts/run_napari_ui.py`

The archived scripts remain here only for backward reference and should not be treated as the current study/review workflow.
